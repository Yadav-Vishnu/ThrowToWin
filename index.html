Dare To Win ?

Game Description : As the game name THROW TO WIN expaline it self, Throw the Dice and win the game .

This game is simply based on the virtual dice. In this you just throw the Dice and watch the number on the dice.

Game Rules :

  • In this game dice is not throw by any player.
  • Dice is throw by the computer of both the playres.
  • Both player will note their dice number.
  • Player who got maximum number on the dice will win the game.
  • If both the player get same dice number then no one be the winner.
